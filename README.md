# Battery_Finder_Program

## Descripsiton
RecursionのProject2「Battery Finder Program」で作成したアプリです。
カメラのブランド名、モデル型番、アクセサリの消費電力を入力すると、
総消費電力に基づいて、カメラセットと互換性のあるすべての電池を表示し、
さらに満充電時にどれほど電池が持続するか表示します。
